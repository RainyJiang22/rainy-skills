# rainy-skills

个人技能skills仓库，搜集/编写Android&Flutter日常开发中用到的 skills，辅助提效

## 安装

### 注册插件市场

在 Claude Code 中运行：

```bash
/plugin marketplace add RainyJiang/rainy-skills
```

### 安装技能

**方式一：通过浏览界面**

1. 选择 **Browse and install plugins**
2. 选择 **rainy-skills**
3. 选择要安装的插件
4. 选择 **Install now**

**方式二：直接安装**

```bash
# 安装指定插件
/plugin install flutter-related@rainy-skills
/plugin install android-related@rainy-skills
/plugin install soft-daily@rainy-skills
```

**方式三：告诉 Agent**

直接告诉 Claude Code：

> 请帮我安装 github.com/RainyJiang22/rainy-skills 中的 Skills
